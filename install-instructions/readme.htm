Clean Installation of Version 2.1.0 Final

A fresh install is for those who have not yet installed a copy of Elite Bulletin Board.

Step 1: Upload content to web directory

Unzip everything from the zip file and upload to some web space that has php/mysql access. You may change the name of the folder if you desire.

Step 2: CHMOD folder

This step is very important. in order for the config file to be created, you must give write access to the folder. You should CHMOD the following folders:

/ebbv2/(777 or 755*)
/ebbv2/install/(777 or 755*)
/ebbv2/uploads/(777 or 755*)
/ebbv2/config.php(777 or 755*)

* - this depends on your web host's setup so ask them which method is prefered.

Step 3: go to install folder

Once all of the contents are uploaded and have all the proper permissions set, open up your web broswer and point it to the path to the install folder(ex: mysite.com/ebbv2/install/). Once there, follow the instructions carefully.

Step 4: Finished

Once you run the installer, you are now finish and may enjoy your new board.

Upgrading from Version 2.0

If you currently have version 2.0 of Elite Bulletin Board installed, please use the version 2.0 upgrade file to keep your current setup.

NOTICE:

When upgrading from Version 2.0, all themes will be removed from the database, they may be added after the install.

Also any modifications done to your board will be undone through the upgrade process so please back up everything before moving on.


Step 1: upload contents to folder

With all of the work done to version 2.1.0, it would best if you deleted either the entire folder or just the contents in that folder.

Once you run the installer, you are now finish and may enjoy your new board.
